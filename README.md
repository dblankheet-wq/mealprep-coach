# Mealprep Coach v1

Een compacte mobiele Next.js-app voor mealprep, boodschappen en afvalanalyse.

## Functies

- Profiel met jouw standaardgegevens
- Weekplanner voor lunch en diner
- Receptendatabase met mealprep-instructies
- Automatische boodschappenlijst
- Picnic-kopieerlijst
- Gewicht bijhouden
- Analyse: tekort, verwacht gewichtsverlies en voorspelling richting doel
- Data-opslag via localStorage in de browser

## Lokaal draaien

```bash
npm install
npm run dev
```

## Deploy op Vercel

1. Upload deze projectmap naar GitHub.
2. Ga naar Vercel.
3. Kies `Import Project`.
4. Selecteer je GitHub-repository.
5. Klik `Deploy`.

## Let op over supermarkt-koppeling

Deze v1 maakt een betrouwbare kopieerlijst voor Picnic. Automatisch winkelmandje vullen is als concept voorbereid, maar vereist een officiële of stabiele supermarkt-API.
